# MiMo Life OS 🚀

نظام إدارة حياة شخصي شامل - مصمم خصيصاً لمحمد (18 سنة، طالب توجيهي صناعي)

## 🎯 المميزات

### 📚 قسم التوجيهي
- تتبع درجات 10 مواد (أكاديمية + مهنية)
- هدف مخصص (400 من 500)
- حساب المعدل التلقائي

### 🔧 مشروع BMS
- تتبع مكونات المشروع
- مراحل التقدم (5 مراحل)
- حالة كل مكون

### 🎓 المنحة والجامعة
- متطلبات المنحة الصينية
- التحضيرات للسفر
- جامعة هونان للتكنولوجيا

### 💼 المشاريع التجارية
- أفكار تجارية
- معرض المشاريع
- حالة كل مشروع

### 📝 الملاحظات
- محرر Markdown
- مجلدات منظمة
- حفظ تلقائي

### 💰 المالية
- تتبع الدخل والمصروفات
- رسم بياني للتوزيع
- رصيد حالي

### 🏥 الصحة واللياقة
- تتبع الماء والنوم
- تمارين رياضية
- سعرات حرارية

### 🔄 العادات اليومية
- تتبع العادات
- Streak لكل عادة
- إحصائيات

### 📚 المكتبة
- قائمة كتب
- جلسات قراءة
- مصادر تعليمية

### 🤖 MiMo AI
- مساعد ذكي شخصي
- محادثات متعددة
- ذاكرة السياق

### 🎮 Gamification
- نقاط خبرة (XP)
- مستويات
- شارات وإنجازات
- وضع المحارب (Hard Mode)

## 🛠️ التقنيات

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **State**: Zustand مع localStorage persistence
- **Charts**: Recharts
- **Animations**: Framer Motion
- **AI**: Zukijourney API (gpt-4o-mini)

## 📦 التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/YOUR_USERNAME/mimo-life-os.git

# الدخول للمجلد
cd mimo-life-os

# تثبيت المتطلبات
bun install

# إنشاء ملف .env
cp .env.example .env

# تشغيل المشروع
bun run dev
```

## 🔑 متغيرات البيئة

```env
ZUKI_API_KEY=your_api_key
ZUKI_BASE_URL=https://api.zukijourney.com/v1
ZUKI_MODEL=gpt-4o-mini
```

## 🚀 النشر على Railway

1. اربط الريبو بـ Railway
2. أضف متغيرات البيئة
3. سيتم النشر تلقائياً
